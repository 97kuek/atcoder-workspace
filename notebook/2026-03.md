### 学んだこと
- `s.count("1")`: 文字列sに含まれる"1"の数を数える。

#### 入力
```python
sys.stdin.read().split()
```
1. `sys.stdin.read()`: 入力されるデータを、プログラムの終わりまですべて一気に読み込む。
2. `.split()`: 読み込んだ巨大な文字列を、スペースや改行がある場所で切り分け、リストに変換する。

```python
list(map(int, input().split()))
```
1. `input()`: ユーザーから"8 12 40"という文字列を受け取る
2. `.split()`: 空白で切り分けて `["8", "12", "40"]` という文字列のリストにする
3. `map(int, ...)`: リストの各要素に `int()` を適用して、整数に変換する
4. `list(...)`: mapオブジェクトをリストに変換する

#### 繰り返し処理
```python
while all(i % 2 == 0 for i in a):
```
1. `(i % 2 == 0 for i in a)`: リストaの各要素に対して、2で割った余りが0であるかを判定する
2. `all(...)`: 全ての要素がTrueであるかを判定する

- ちなみに`any()`もあり、少なくとも一つでもTrueであればTrueを返す。

#### リスト内包表記
既存のリストを加工して、新しいリストを作る書き方。基本形としては、
`[新しい要素にする計算 for 取り出した要素 in 元のリスト]`
本来、
```python
new_a = []
for i in a:
    new_a.append(i//2)
a = new_a
```
と書くところを、リスト内包表記を使えば
```python
a = [i // 2 for i in a]
```
で1行で書くことが出来る。
